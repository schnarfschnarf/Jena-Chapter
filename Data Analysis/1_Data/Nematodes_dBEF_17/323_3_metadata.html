Data Processing *
Data preparation ? which software? if in R, which package? The dataset provides: 1. Nematode taxa richness = taxa per sample 2. Shannon Diversity H’: Shannon or Shannon- Wiener index is defined as H=−sum pi log(b)pi, where pi is the proportional abundance of species i and b is the base of the logarithm. We use b=10. The calculation with the software Marcel is using and the function in vegan diversity(com.mat, index = "shannon", MARGIN = 1, base = 10) give the same results. 3. Shannon Hmax: Hmax = log10(S), S = species richness per sample. Some people calculate Hmax as ln(S) 4. Shannon J’ (Species evenness): H’/Hmax 5. Simpson Diversity (D): D= ((∑n(n-1))/(N(N-1))), with n = the number of each species, and N the total number of species. In R, D= ∑pi^2, with pi is the frequency of each species. Further R returns the simpson index as 1-D, in vegan the formula is: diversity(com.mat, index = "simpson") Note: D is calculated differently from the version in vegan. 6. Simpson Diversity (1/D). Note: 1/D is calculated differently from the version in vegan. 7. Dmax: =1-(1/S) 8. E: =D/(1-(1/S)) Note: 1/D is calculated differently from the version in vegan. Interpretation: The Shannon index is sensitive to rare species and the Simpson index weights common species.
